# DIGIWAKE
Conception réveil

● EF1 : Le système doit compter le temps et afficher l’heure.

● EF2 : Un affichage doit permettre d’apprécier un battement à la seconde près.

● EF3 : L’utilisateur doit pouvoir régler l’heure, ainsi que l’heure de réveil, et activer ou non le réveil.

● EF4 : L’heure doit pouvoir être affichée selon un format 24h ou 12h AM/PM.

● EF5 : L’utilisateur doit être averti par le biais d’une mélodie à l’heure du réveil.

● EF6 : L’utilisateur doit pouvoir éteindre la mélodie avant la fin.
